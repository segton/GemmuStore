A first-person 3D shop simulator built in C++ with Raylib, Jolt Physics, ImGui, and GLSL.

Key systems:
- Customer AI and POI navigation
- Item pickup, inspection, placement
- Seller/buyer transaction flow
- Day-based shop loop
- ImGui scene editor tools
- GLB scene prop workflow
- Jolt collider setup and debug drawing
- Custom shaders, shadows, render targets, and graphics settings
